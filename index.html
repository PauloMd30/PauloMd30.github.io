Gestão de Produtos

Esta aplicação foi desenvolvida com Python (Flask) e tem como objetivo gerenciar produtos com base em sua validade, oferecendo funcionalidades automatizadas de controle e notificação.
Funcionalidades principais: Cadastro, edição, listagem e exclusão de produtos com campos como nome, marca, código de barras e data de validade. Interface web integrada usando templates Jinja2 para interação com o sistema. Validação automática de datas no backend para garantir integridade dos dados. Consulta de produtos com validade próxima (até 7 dias) para facilitar o controle de estoque. Exclusão automática de produtos vencidos com verificações diárias agendadas (via APScheduler).

Portal de Notícias

Desenvolvi uma aplicação web completa para gerenciamento de notícias, utilizando Node.js, Express, MySQL e template engine (como EJS). A solução permite exibir, cadastrar e validar notícias por meio de um painel administrativo, oferecendo uma experiência eficiente tanto para usuários quanto para administradores. Principais funcionalidades: Listagem dinâmica de notícias com ordenação por data; Exibição de notícia individual com base no ID via query string; Validação de formulário com Express Validator, garantindo integridade dos dados; Persistência de dados em banco de dados MySQL utilizando DAO (Data Access Object); Tratamento de erros e feedbacks ao usuário, com renderização de views personalizadas; Arquitetura modular, com separação clara entre controladores, modelos e rotas. Destaques técnicos: Uso assíncrono com async/await para garantir performance e legibilidade; Implementação do padrão DAO para encapsular a lógica de acesso a dados; Validações server-side robustas para garantir qualidade dos dados inseridos; Integração com views dinâmicas (como EJS/Pug) para renderização fluida no front-end.
